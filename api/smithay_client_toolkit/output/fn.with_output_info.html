Access the info associated with this output

The provided closure is given the OutputInfo as argument, and its return value is returned from this function.

If the provided WlOutput has not yet been initialized or is not managed by SCTK, None is returned.

If the output has been removed by the compositor, the obsolete field of the OutputInfo will be set to true. This handler will not automatically detroy the output by calling its release method, to avoid interfering with your logic.
