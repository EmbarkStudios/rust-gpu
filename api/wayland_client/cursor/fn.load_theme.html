[−][src]Function wayland_client::cursor::load_theme

pub fn load_theme(name: Option<&str>, size: u32, shm: &WlShm) -> CursorTheme

Attempts to load a cursor theme.

If no name is given or the requested theme is not found, the default theme will be loaded.

Other arguments are the requested size for the cursor images (ex: 16) and a handle to the global WlShm object.

Panics

  • Panics if the wayland-cursor lib is not available (see is_available() function) in this module.
  • Panics in case of memory allocation failure.
  • Panics if name contains an interior null.
