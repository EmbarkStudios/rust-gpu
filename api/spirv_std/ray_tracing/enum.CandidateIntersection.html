#[repr(u32)]
pub enum CandidateIntersection {
    Triangle,
    AABB,
}
Expand description

Describes the type of the intersection which is currently the candidate in a ray query, returned by RayQuery::get_candidate_intersection_type.

Variants

Triangle

A potential intersection with a triangle is being considered.

AABB

A potential intersection with an axis-aligned bounding box is being considered.
