Wrapper for handling the difference between single-version and multi-version output, which aren’t expressible in pretty::Fragment.

Each node has definitions “tagged” with an usize indicating the number of versions that share that definition, aka “repeat count” (i.e. “repeat counts” larger than 1 indicate deduplication).
