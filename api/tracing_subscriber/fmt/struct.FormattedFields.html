A formatted representation of a span's fields stored in its extensions.

Because FormattedFields is generic over the type of the formatter that produced it, multiple versions of a span's formatted fields can be stored in the Extensions type-map. This means that when multiple formatters are in use, each can store its own formatted representation without conflicting.
